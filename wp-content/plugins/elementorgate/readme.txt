A collection of power tools for Elementor Editor - command palette, typography previews, keyboard navigation, and more.

== Description ==

ElementorGate extends Elementor page builder with developer productivity tools designed to speed up your workflow.

**Features:**

* **Command Palette (Cmd+K)** - Raycast-style command launcher with 25+ commands organized by category
* **Typography Preview** - Hover tooltips showing global typography settings
* **Keyboard Navigation** - Arrow-key navigation between elements with Option modifier
* **CSS/ID Visualizer** - Displays custom CSS classes and IDs as visual labels
* **Wrap/Unwrap Containers** - Figma-style Cmd+G to wrap elements, Cmd+Shift+G to unwrap
* **Template Previews** - Thumbnail preview functionality for template library
* **Spacing Measure** - Visual margin/padding display

**Requirements:**

* WordPress 6.0 or higher
* Elementor plugin (free or Pro)
* PHP 7.4 or higher

== Installation ==

1. Upload the `elementorgate` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this work with Elementor Free? =

Yes, this plugin works with both Elementor Free and Elementor Pro.

= How do I access the command palette? =

Press Cmd+K (Mac) or Ctrl+K (Windows) while in the Elementor editor.

= How do I wrap elements in a container? =

Select one or more elements and press Cmd+G (Mac) or Ctrl+G (Windows).
